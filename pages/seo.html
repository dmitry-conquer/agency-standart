SEO Best Practices & Optimization

Meta tags, structured data, URL structure, content optimization, and technical SEO standards.

Proper meta tags and structured data improve search engine visibility and click-through rates.

HTML Meta Tags 
<!-- Essential Meta Tags -->
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<meta name="description" content="Clear, concise description (150-160 characters)">
<meta name="keywords" content="relevant, keywords, here">

<!-- Open Graph (Social Media) -->
<meta property="og:title" content="Page Title">
<meta property="og:description" content="Page description">
<meta property="og:image" content="https://example.com/image.jpg">
<meta property="og:url" content="https://example.com/page">
<meta property="og:type" content="website">

<!-- Twitter Card -->
<meta name="twitter:card" content="summary_large_image">
<meta name="twitter:title" content="Page Title">
<meta name="twitter:description" content="Page description">
<meta name="twitter:image" content="https://example.com/image.jpg">
JSON-LD Structured Data 
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "WebPage",
  "name": "Page Title",
  "description": "Page description",
  "url": "https://example.com/page",
  "inLanguage": "en-US",
  "isPartOf": {
    "@type": "WebSite",
    "name": "Site Name",
    "url": "https://example.com"
  }
}
</script>
  • ✓ Always include unique meta descriptions (150-160 characters)
  • ✓ Use Open Graph tags for social media sharing
  • ✓ Implement JSON-LD structured data for rich snippets
  • ✓ Validate structured data with Google Rich Results Test

Clean URLs and proper sitemap configuration help search engines crawl and index your site effectively.

URL Best Practices 
// Good URLs
https://example.com/blog/seo-best-practices
https://example.com/products/laptop-2024
https://example.com/about/team

// Bad URLs
https://example.com/page?id=123&cat=abc
https://example.com/index.php?p=456
https://example.com/2024/01/15/post-title-12345
robots.txt 
User-agent: *
Allow: /
Disallow: /admin/
Disallow: /wp-admin/
Disallow: /wp-includes/
Disallow: /private/

Sitemap: https://example.com/sitemap.xml
  • ✓ Use descriptive, keyword-rich URLs (avoid IDs and parameters)
  • ✓ Keep URLs short and readable (3-5 words maximum)
  • ✓ Use hyphens to separate words, not underscores
  • ✓ Create and submit XML sitemap to Google Search Console
  • ✓ Configure robots.txt to guide crawlers

Optimize content structure, headings, and keywords for better search rankings.

Heading Structure 
<h1>Main Page Title (One per page)</h1>
<h2>Section Heading</h2>
<h3>Subsection Heading</h3>
<h4>Minor Heading</h4>

<!-- Never skip heading levels -->
<!-- Bad: h1 → h3 (skipping h2) -->
<!-- Good: h1 → h2 → h3 -->
  • ✓ Use one H1 per page with primary keyword
  • ✓ Maintain proper heading hierarchy (H1 → H2 → H3)
  • ✓ Include keywords naturally in content (avoid keyword stuffing)
  • ✓ Use alt text for all images with descriptive keywords
  • ✓ Create internal links to related content
  • ✓ Optimize content length (aim for 1000+ words for blog posts)

Technical SEO ensures your site is fast, mobile-friendly, and properly indexed by search engines.

Canonical Tags 
<!-- Prevent duplicate content issues -->
<link rel="canonical" href="https://example.com/page">

<!-- For paginated content -->
<link rel="prev" href="https://example.com/page/2">
<link rel="next" href="https://example.com/page/4">
Performance Optimization 
// Image optimization
<img src="image.webp" 
     alt="Description" 
     loading="lazy" 
     width="800" 
     height="600">

// Preload critical resources
<link rel="preload" href="/fonts/main.woff2" as="font" type="font/woff2" crossorigin>
<link rel="preload" href="/styles/critical.css" as="style">
  • ✓ Ensure site is mobile-responsive (mobile-first design)
  • ✓ Optimize page speed (target <3s load time)
  • ✓ Use HTTPS (SSL certificate required)
  • ✓ Implement canonical tags to prevent duplicate content
  • ✓ Enable GZIP compression and browser caching
  • ✓ Fix broken links and 404 errors
  • ✓ Monitor Core Web Vitals (LCP, FID, CLS)
