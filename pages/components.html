Components

Reusable Component System

Standardized components for popups, tabs, accordions, and forms.

All components use data-attributes for initialization. No class-based selectors.

Component Pattern 
// All components follow this pattern:
// 1. HTML with data-attributes
// 2. JavaScript class initialization
// 3. CSS with BEM naming

// HTML
<div class="accordion" data-accordion="faq">
  <button class="accordion__header" data-accordion-trigger>
    Question 1
  </button>
  <div class="accordion__panel" data-accordion-panel>
    Answer 1
  </div>
</div>

// JavaScript
document.querySelectorAll('[data-accordion]').forEach(container => {
  new Accordion(container);
});

// CSS
.accordion { }
.accordion__header { }
.accordion__panel { }
.accordion__panel--active { }
  • ✓ data-modal - Modal dialogs
  • ✓ data-tabs - Tab navigation
  • ✓ data-accordion - Accordion panels
  • ✓ data-dropdown - Dropdown menus
  • ✓ data-slider - Image/content sliders
  • ✓ data-form - Form validation
